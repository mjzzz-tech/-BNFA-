# 原文说明

该项目为文章《面向鸟鸣识别的背景噪声过滤算法设计与实现》的代码实现。该算法使用Deep Complex U-net对含噪声的鸟鸣音频进行首次去噪，接着采用融合了CBAM注意力机制的ResNet18作为深度特征提取网络在特征提取过程中过滤噪声，可以有效提取鸟鸣深度特征，随后经过归一化的特征送入浅层分类器进行识别。鸟鸣识别作为生态监测的重要手段，具有重要的研究意义和应用前景。
